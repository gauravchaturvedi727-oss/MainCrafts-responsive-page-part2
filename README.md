# 🚀 MainCrafts - Responsive Business Website

A modern, responsive multi-page business website built using **HTML, CSS, and JavaScript**. The project demonstrates responsive web design, clean UI, and client-side form validation.

---

## 📌 Features

- ✅ Fully Responsive Design
- ✅ Modern Navigation Bar
- ✅ Hero Section
- ✅ About Page
- ✅ Contact Page
- ✅ Services / Features Section
- ✅ Contact Form Validation
- ✅ Smooth Scrolling
- ✅ Mobile Navigation Menu
- ✅ Hover Animations
- ✅ Professional Footer
- ✅ Clean & Reusable Code

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
